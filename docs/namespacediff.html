std::vector<IndexT> diff::myers_diff_trace ( ARR  orig,
ARR  updated

Return the diff trace, the flattened representation of 2-D array <2 * (orig.size + updated.size) + 1, min_edits> each row in the trace matrix are the minimum costs at every step each row's origin starts at index orig.size + updated.size zeros in each row represent unreachable states at that step This function is a helper function for myers_diff, but can use in diagnosis
